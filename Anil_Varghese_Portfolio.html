Gating — governance for production AI

Observability at retail scale

Observability
Lowe's

Built and ran observability — Prometheus, Jaeger, Elasticsearch, Thanos — across 200+ microservices serving 17M+ weekly users, holding 99.9% uptime through peak retail events.

When a pricing lookup API began failing intermittently, the trail led to an uninstrumented DB call, an overly broad exception handler swallowing errors, and a partial Node 12/14 version split causing inconsistent MongoDB behavior. Fix: a fleet-wide Node version rollback, new tracing spans, and SLA-based alerting on customer-facing DB calls.

Scale: 17M+ weekly users
Reliability: 99.9% uptime, peak events
